Methods

create(msg) → {boolean}

Send a new message
Parameters:
Name Type Description
msg Object The message to be sent
Properties
Name Type Attributes Description
pattern string The message pattern. One of FCFS, STD, FYI
members Array Array of identifiers to send the message to
content string The body of the message
maxAccepts number <optional>
 If the pattern if FCFS this sets the number of 'Accepts' to consider the message complete, if the pattern is FCFS and this parameter is not specified it will default to 1
Returns:
returns if the request was successfully made, a successful request may still result in an error being emitted. A successful message will emit the messages id on the 'new' topic
Type
boolean

update(m, action, dataopt) → {boolean}

Update an existing message
Parameters:
Name Type Attributes Description
m Object The message
action string The action to perform on the message. Must be one of the allowed actions in the message.
data string <optional>
 Action specific data. If the Action is 'FORWARD' then data must contain the UUID of the member the message is being forwarded to If the Action is 'DELEGATE' then the data must contain the UUID of the member the message is being delagated to
Returns:
returns if the request was successfully made, a successful request may still result in an error being emitted. A successful message will emit the messages id on the 'update' topic
Type
boolean
