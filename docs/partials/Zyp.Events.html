Methods

on(type, callback, context)

Sets an application handers for ZypHub Events.
Arguments:
  1. type (string)  The event to be handled. Valid events are: login, error, websocket
  2. callback (EventHandlerCallback)  The handler function
  3. context (Object)  Optional A context object that will be also passed to the hander with the event data
