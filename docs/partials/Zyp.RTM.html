create(msg) → boolean

Send a new message
Arguments:
  1. msg (Object)  The message to be sent
    Properties
    1. pattern (string)  The message pattern. One of FCFS, STD, FYI
    2. members (Array)  Array of identifiers to send the message to
    3. content (string)  The body of the message
    4. [maxAccepts] (number)  If the pattern if FCFS this sets the number of 'Accepts' to consider the message complete, if the pattern is FCFS and this parameter is not specified it will default to 1

update(m, action, dataopt) → boolean

Update an existing message
Arguments:
  1. m (Object)  The message
  2. action (string)  The action to perform on the message. Must be one of the allowed actions in the message.
  3. [data] (string)  Action specific data. If the Action is 'FORWARD' then data must contain the UUID of the member the message is being forwarded to If the Action is 'DELEGATE' then the data must contain the UUID of the member the message is being delagated to
